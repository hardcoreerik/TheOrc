## What We're Building

A standalone **Python GUI desktop application** that:
- Takes any topic as input
- Scrapes multiple web sources (Wikipedia, DuckDuckGo, Reddit, raw URLs)
- Aggregates and deduplicates content
- Sends it to a local Ollama model to generate structured HOW-TO docs
- Saves everything as organized Markdown research files
- Lets the user pick any installed Ollama model

## Project: OrcResearcher

### Stack
| Layer | Choice | Why |
|-------|--------|-----|

| Scraping | requests + beautifulsoup4 | Standard, reliable |

- [ ] `scraper.py`
  - [ ] `fetch_wikipedia(topic)` → extract main article text, first 3 sections

  - [ ] `fetch_url(url)` → generic: strip HTML, return clean text
  - [ ] `deduplicate(texts)` → remove near-duplicate paragraphs

### Phase 3 — Ollama Integration ✅ Target
- [ ] `ollama_client.py`

  - [ ] Prompt template: "You are a research assistant. Given the following sources on '{topic}', write a clear HOW-TO guide with: Overview, Prerequisites, Step-by-Step, Key Concepts, and Resources sections."
