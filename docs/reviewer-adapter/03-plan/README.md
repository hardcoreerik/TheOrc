# Plan

The canonical plan for the reviewer adapter is the consolidated guide:
[`../../REVIEWER_ADAPTER_GUIDE.md`](../../REVIEWER_ADAPTER_GUIDE.md).

It holds the two-stage training architecture (public-corpus SFT → TheOrc-capture
LoRA), the three-tier source strategy, the phase gate, the eval metric, and the
trust-promotion ladder.

This folder exists for plan *deltas* — when a planning decision changes, record
the dated decision here and update the canonical guide to match. As of
2026-06-13 there are no deltas; the guide is current.
